## [v3.0] — April 2026

### Added
- **Cognitive Domain (CD) Stratum** — 8th BGC domain formally integrated across all actor profiles, weight schedules, seam analysis, and scenario inject library
- **CD-NOF Library** — eight Normalized Operational Factors (CD-NOF-01 through CD-NOF-08) with measurement definitions and Winter Storm 2030 scenario applications
- **Gray Zone Governance Framing** — primary analytical orientation updated throughout: platform context, gap statement, disruptive positioning, and thesis quote
- **Governance Lab Tab** — document upload (PDF, TXT, DOCX, MD), E-IAIG-HT gray zone analysis via Anthropic API, Knowledge Base with category filtering
- **Pre-loaded Knowledge Base entries** — Svalbard Treaty (1920) and NATO Article 5 Washington Treaty (1949)
- **KPIs / Success Criteria Tab** — five approaches to success criteria design, five generic SAS-219 KPIs, five WinterStorm2030-specific KPIs, live demo of all five KPI outputs
- **Two new CD governance seams** — CD Doctrine Asymmetry (Russia/Iceland, 44 pts) and Alliance Cognitive Coherence Gap (Finland-Sweden/Iceland, 26–28 pts)
- **Governance Lab nav entry** in Overview navigation guide table
- **CD column** in Actor Analysis table and Demo actor table

### Changed
- All actor g-GWC scores revised to reflect 8-domain equation (minor adjustments, <2 pts)
- Actor toggle cards now display CD scores alongside g-GWC
- Framework Note in Agentic Foresight panel updated — E-AGPO-HT reference corrected to E-IAIG-HT
- Section subtitle in Actor Analysis panel corrected from E-AGPO-HT to E-IAIG-HT
- Governance Lab system prompt corrected from E-AGPO-HT to E-IAIG-HT

### Fixed
- Four E-AGPO-HT framework label errors corrected to E-IAIG-HT throughout platform
- WS-KPI-1 Governance Baseline Coverage measurement method corrected from "Auracelle Bach platform" to "Auracelle WinterStorm2030"
